## 6. AI Prompt Journal

---

### Prompt I — Introduction to JavaScript
**Prompt used:**
> "I am a beginner in JavaScript and I am interested in
> learning the language. What is JavaScript and what makes
> it unique compared to other programming languages?
> Where is it used, how popular is it, what are its pros
> and cons, and if I were to learn JavaScript today how
> would I go about doing that?"

**AI Response Summary:**
The AI explained that JavaScript is the most widely used
programming language in the world. It runs directly in
the browser without any installation and is used
everywhere from websites to mobile apps to servers.
It also gave a clear learning path starting from basics
like variables and functions all the way to building
real projects.

**Helpfulness:** ⭐⭐⭐⭐⭐
Very helpful — gave a full picture of what JavaScript is
and why it is worth learning before writing a single
line of code.

---

### Prompt II — Core Concepts
**Prompt used:**
> "I have just finished learning the definition of
> JavaScript. What are the core concepts a complete
> beginner must understand before diving deep into
> JavaScript? Which concept should I learn first and
> why? Can you give a simple real world analogy for
> each concept?"

**AI Response Summary:**
The AI identified 7 core building blocks — variables,
data types, functions, conditionals, loops, DOM
manipulation and error handling. It explained each
one using simple real world analogies which made them
easy to understand before seeing any code.

**Helpfulness:** ⭐⭐⭐⭐⭐
Extremely helpful — the analogies made abstract
concepts feel familiar and easy to grasp.

---

### Prompt III — Setup & Installation
**Prompt used:**
> "I have just finished learning the definition of
> JavaScript and its core concepts. I want to now know
> how to install JavaScript in order to start coding.
> What are the system requirements, what tools do I
> need and how do I set up my development environment
> step by step?"

**AI Response Summary:**
The AI clarified that JavaScript does not need to be
installed separately since it already runs in the
browser. It recommended VS Code as the code editor
and the Live Server extension to run code locally.
It also gave a step by step setup guide.

**Helpfulness:** ⭐⭐⭐⭐⭐
Very helpful — cleared up a common misconception that
JavaScript needs a separate installation like Python
or Java.

---

### Prompt IV — Common Beginner Errors
**Prompt used:**
> "I have just finished setting up JavaScript on my PC.
> What are the most common errors beginners make in
> JavaScript, why do they happen, how do I identify
> and fix each one, and what good habits can I develop
> early to avoid these errors?"

**AI Response Summary:**
The AI listed the most common beginner errors including
ReferenceError, SyntaxError, TypeError and the common
confusion between = and ===. It explained why each
error happens and gave clear fixes and good habits
to develop early.

**Helpfulness:** ⭐⭐⭐⭐
Very helpful — knowing the errors before encountering
them saved a lot of debugging time later.

### Prompt V — Hello World
**Prompt used:**
> "I have just finished learning the definition of
> JavaScript, its core concepts, how to set it up and
> common errors to avoid. Walk me through writing a
> simple Hello World program in JavaScript step by
> step. Explain what each line of code does and show
> me how to run it."

**AI Response Summary:**
The AI walked through creating the HTML file, linking
the JavaScript file and using
document.getElementById().innerHTML to display Hello
World on the page. It explained what each line does
in simple terms and showed how to verify it works
using Live Server.

**Helpfulness:** ⭐⭐⭐⭐⭐
Very helpful — the step by step explanation made it
easy to understand how JavaScript connects to HTML
for the first time.

---


### Prompt VI — DOM Manipulation
**Prompt used:**
> "I have just finished writing my first Hello World
> program using JavaScript. What is the DOM and why
> is it important? How do I select and modify elements
> on a webpage using JavaScript? How do I make a
> webpage respond to user actions like button clicks?
> Can you show me a simple example I can try myself?"

**AI Response Summary:**
The AI explained that the DOM is a live map of the
webpage that JavaScript can read and change at any
time. It showed how to use getElementById to find
elements, innerHTML to change content and
addEventListener to respond to user clicks. It also
gave a simple button click example to try.

**Helpfulness:** ⭐⭐⭐⭐⭐
Very helpful — understanding the DOM was the key to
making the To-Do List app work. Without it none of
the tasks could appear on the page.

---

### Prompt VII — Building the Project
**Prompt used:**
> "I have just finished learning JavaScript basics
> including core concepts, setup, common errors, my
> first Hello World program and basic DOM manipulation.
> Suggest a simple beginner friendly project I can
> build, walk me through building it step by step,
> explain how the core concepts I have learned are
> being applied and show me how to test it."

**AI Response Summary:**
The AI suggested a To-Do List app as the perfect
beginner project since it uses all 7 core concepts
in one application. It walked through building it
step by step — starting with the HTML structure,
then the CSS styling and finally the JavaScript
logic. It also provided a testing checklist to
verify everything works correctly.

**Helpfulness:** ⭐⭐⭐⭐⭐
Extremely helpful — the step by step approach made
it easy to follow and the explanations of how each
concept was being used helped connect theory
to practice.
