## Technology Selected: JavaScript

## AI Prompt Plan — Using Claude AI

Here are the prompts I will use throughout my JavaScript 
learning journey:

### I. What is JavaScript?
What is JavaScript, where is it used, real world examples, 
why it is preferred today, pros and cons, and how it 
compares to other languages.

### II. Core Concepts
What are the core concepts a beginner should know before 
diving deep into JavaScript?

### III. Setup & Installation
What are the system requirements for JavaScript and how 
do I install and set up my development environment?

### IV. Common Beginner Errors
What are the most common errors beginners make in 
JavaScript and how do I fix them?

### V. First Program
How do I write a simple program using JavaScript?

### VI. DOM Manipulation
How do I manipulate the DOM in JavaScript?

### VII. Building a Project
How do I build and test a simple JavaScript project using 
the concepts I have learned?
