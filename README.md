# JavaScript Beginner Toolkit 🚀
### By DIANA KITONYI

---

## Overview
This repository is a beginner friendly JavaScript toolkit 
built as part of my Moringa School Capstone Project.
It uses AI prompts to guide a complete beginner through 
learning JavaScript from scratch and building a simple 
working project.

---

## What is in this Repo

| File | Description |
|---|---|
| `README.md` | Overview of the repository |
| `toolkit1.md` | JavaScript introduction, core concepts and Hello World guide |
| `toolkit2.md` | Step by step guide to building a To-Do List app |
| `index.html` | Main webpage for the To-Do List project |
| `style.css` | Styling for the To-Do List project |
| `script.js` | JavaScript code for the To-Do List project |

---

## Who is this for?
This toolkit is designed for:
- Complete beginners with no coding experience
- Anyone who wants to learn JavaScript from scratch
- Students looking for a structured JavaScript learning path

---

## How to Run the Project
1. Clone this repository or download the zip file
2. Open the project folder in VS Code
3. Right click `index.html` and click **Open with Live Server**
4. The To-Do List app will open in your browser
