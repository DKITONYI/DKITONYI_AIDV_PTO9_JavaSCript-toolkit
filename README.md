## 📌 Overview  
This repository is a **beginner-friendly JavaScript toolkit** created as part of my *Moringa School Capstone Project*.  

It is designed to help beginners:
- Learn JavaScript from scratch  
- Practice concepts with simple examples  
- Build a real project step-by-step

## 📖 How to Use This Toolkit  

To get the most out of this repository, follow this learning path:

1. Start with the **`learn-JavaScript`** folder  
2. Open and go through `toolkit1.md` step-by-step  
3. Practice using the examples provided in the `examples/` folder  
4. Make sure you understand the basics before moving on  

---
## How to Run the Project  

1. Clone or download the repository  
2. Open the folder in **VS Code**  
3. Navigate to the `todo-app` folder  
4. Open `index.html` with **Live Server**  
5. Start using your To-Do List app 🎉
